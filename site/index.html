Interactive STAPL study material

Learn STAPL by mapping it to systems, C++, and the papers that built the project.

This site is centered on the hardest conceptual jump for most students: moving from ordinary single-process C++ reasoning to distributed-memory parallel reasoning with STAPL.

What this teaches

  • Why STAPL is not just "STL but faster".
  • How locations, views, containers, and algorithms change memory reasoning.
  • How this connects to data structures, OS, architecture, parallel programming, and core C++.
  • How the public STAPL papers motivate the abstractions you see in the code and docs.
